## Estrutura do projeto

```
ceciliachaves/
├── index.html          → Home
├── comercial.html      → Página comercial (patrocinadores)
├── bolos.html          → Bolos sob encomenda
├── cursos.html         → Cursos online
├── style.css           → Estilos compartilhados (todas as páginas)
├── script.js           → JavaScript (menu mobile)
└── assets/
    ├── images/         → Fotos
    └── logos/          → Logos das sub-marcas
```

## Como editar

### Edições rápidas de texto
1. Abra o arquivo `.html` da página que quer editar
2. Edite o texto entre as tags HTML (não mexa nas tags em si)
3. Salve, faça commit no GitHub Desktop e push
4. Em ~30 segundos a alteração vai estar no ar

### Trocar uma imagem
1. Coloque a nova imagem na pasta `assets/images/` com o mesmo nome da antiga
2. Commit + push
3. Pronto

### Mudar cor ou estilo
- Todas as cores estão em `style.css` no topo (variáveis CSS)
- Estilos específicos de cada página estão dentro do próprio `.html`
